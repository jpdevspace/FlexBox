This is my personal guide to Flexbox

The Flexible Box Module, more commonly known as flexbox, was designed as a one-dimensional layout Module and a method for space distribution between items.

The main axis

    The main axis will run horizontally when you define the flex-direction property as:
  • row
  • row-reverse
    The main axis will run vertically when you define the flex-direction property as:
  • column
  • column-reverse
